Introduction

The assignment centers on sentiment analysis, a technique in text mining that attempts to assess the emotional tone of written text by evaluating the words it contains against a predefined set of scored or categorized words called a lexicon. Chapter 2 walks through how to implement sentiment analysis within the tidytext framework, using AFINN, Bing, and NRC. The task at hand is twofold process where first, reproduce the core example from that chapter inside R. The goal is not just to run the code but to understand and explain how the findings from the new corpus and new lexicon compare to what the original example produced.

tidy_books <- austen_books() %>%
  group_by(book) %>%
  mutate(
    linenumber = row_number(),
    chapter = cumsum(str_detect(text, 
                                regex("^chapter [\\divxlc]", 
                                      ignore_case = TRUE)))
  ) %>%
  ungroup() %>%
  unnest_tokens(word, text)

Conclusion

This analysis demonstrated how sentiment analysis can be implemented using tidy data principles in R, first by reproducing the core example from Chapter 2 of the book using Jane Austen’s novels, and then by extending it to a stylistically different corpus and an additional lexicon. Applying the Bing and Syuzhet lexicons to Bram Stoker’s Dracula revealed a narrative arc that trends considerably more negative than anything found in Austen’s works, which aligns with the gothic horror genre of the novel. The comparison between Bing and Syuzhet also highlighted an important methodological point: lexicons do not always agree on the emotional weight of the same text, and the differences are not random. Syuzhet, having been developed and validated specifically on narrative fiction, captures the emotional structure of Dracula in a way that feels more calibrated to the text, while Bing, built on a broader and more modern vocabulary, applies a somewhat blunter instrument to 19th century prose. The tidy text approach proved to be a flexible and readable framework for exploring how emotional tone shifts across the arc of a narrative, and the addition of a domain specific lexicon like Syuzhet meaningfully enriched the analysis beyond what the three built in tidytext lexicons alone could offer.
