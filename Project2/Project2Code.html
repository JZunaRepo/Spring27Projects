Introduction

Project 2 explores data tidying and transformation in R using three distinct wide format datasets pulled from real-world sources: CONMEBOL 2026 FIFA World Cup qualification standings, a global list of cities ranked by average monthly precipitation, and historical NBA per game statistics from Basketball Reference. Each dataset shares the same structural problem in that key variables are spread across multiple columns rather than being organized into a clean key-value format, making them difficult to analyze directly. The goal of this assignment is to apply the tidyr and dplyr packages to reshape all three datasets from wide to tidy long format, normalize variable structures, handle missing or inconsistent values, and then perform a meaningful analysis on each cleaned dataset. Together these three sources offer a range of contexts, from sports and climate to international soccer, that will demonstrate how the same core tidying principles can be applied across very different data environments.

Conclusions

In this project I applied a consistent data tidying and transformations pipeline across three structurally distinct wide-format datasets using the tidyr and dplyr packages. For each dataset, the raw data was first preserved in its original wide format as a csv file before any transformation took place, establishing a clean and reproducible starting point. From there, pivot_longer() was used to collapse wide columns into tidy long format structures, rename_with(tolower) was applied to normalize variable naming conventions, and any missing values were handled deliberately, non-numeric entries, and structural inconsistencies unique to each source.

The CONMEBOL data required character cleaning and the separation of a combined standings and match result matrix into two independent tidy tables. The precipitation dataset involved comma formatted numeric values and the conversion of month abbreviations into an ordered factor to support correct montly visualization. The NBA dataset presented the most complex cleaning challenge, requiring the detection and removal of repeated table header rows and a reasoned decision to retain historical NAs rather than discard decades of not available three point era data. Across all three datasets, the final analysis was performed exclusively on the tidy output, with ggplot2 visualizations and summary tables used to interpret each result showing how a structured tidying workflow enables meaningful and reproducible analysis.
